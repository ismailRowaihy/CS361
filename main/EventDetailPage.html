Web Development

Overview

A hands-on workshop that takes you from a blank file to a working website. You will learn how to structure a page with HTML, style it with CSS, and add server-side logic with PHP. Every session ends with something you built yourself, and the final week is spent turning those pieces into one complete project you can put in a portfolio.

What You Will Learn

  • HTML structure, semantic tags, forms and tables
  • CSS styling, the box model, Flexbox and responsive layouts
  • PHP basics: variables, conditions, loops and functions
  • Handling form submissions and connecting to a MySQL database
  • Publishing your finished site so others can visit it

Requirements

  • A laptop with at least 4 GB of RAM
  • A code editor installed (VS Code is recommended)
  • Basic computer literacy — no prior coding experience needed
  • Willingness to attend all sessions and finish the weekly tasks

Who Should Attend

University students from any major who want a practical introduction to building for the web, whether for coursework, a personal project, or as a first step toward a development career.

Details

  • Duration: 3 weeks, 2 sessions per week
  • Session length: 2 hours
  • Format: In person, hands-on
  • Seats: Limited to 25 students
